# Getting Help with Assura

Thank you for using Assura! Here is how to get help depending on your situation.

## Bug Reports and Feature Requests

Please open an issue on GitHub:

- [Bug reports](https://github.com/assura-lang/assura/issues/new?template=bug_report.yml)
- [Feature requests](https://github.com/assura-lang/assura/issues/new?template=feature_request.yml)

Search [existing issues](https://github.com/assura-lang/assura/issues) first to
avoid duplicates.

## Questions and Community Help

For general questions, usage help, and community discussion, use
[GitHub Discussions](https://github.com/assura-lang/assura/discussions).
If Discussions is not yet enabled on the repository, it will be available soon.
In the meantime, feel free to open an issue with the "question" label.

## Contributing

See [CONTRIBUTING.md](CONTRIBUTING.md) for instructions on setting up the
development environment, building, testing, and submitting pull requests.

## Security Vulnerabilities

If you discover a security vulnerability, please do **not** open a public issue.
Follow the responsible disclosure process described in [SECURITY.md](SECURITY.md).
